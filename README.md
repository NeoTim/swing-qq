# swing-qq
使用java-swing开发的“局域网聊天工具”，(课程设计)只是功能有了，离使用还有些距离。

主题：基于BMP图像传输的信息隐藏算法设计与实现

此代码是多年前课程设计，今日整理学生时期硬盘找到的，遂发到这里。

## 功能
1. 局域网自动发现服务端ip，并进行连接
2. 群聊、私聊
3. 发送加密文件
